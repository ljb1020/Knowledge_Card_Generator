# UI_SPEC.md

## 目标

UI 风格定为：**浅色、专业、克制、结构化的桌面工具台**。  
不要做成官网、作品集、营销页或炫技界面。

## 布局

主页面固定三栏：

- 左栏：输入与操作，宽 `320px`
- 中栏：结构化编辑，宽 `420px`
- 右栏：实时预览，自适应

另有导出页：

- `/export/:jobId`

不改三栏结构，不做复杂导航，不做花哨动效。

## 视觉

- 页面背景：`#F8FAFC`
- 面板背景：`#FFFFFF`
- 主文字：`#0F172A`
- 次级文字：`#475569`
- 主色：`#2563EB`
- 边框：`#E2E8F0`

字体只用系统字体栈，不引入外部字体。

## 风格约束

- 留白充分，信息清晰，强调“编辑”和“预览”
- 右侧预览是主视觉区域，但仍然是工具预览，不是展示海报页
- 预览与导出必须复用同一套卡片组件和样式
- 第一版只允许一套浅色主题

## 组件风格

- 按钮、输入框、面板风格统一
- 圆角和阴影保持克制
- 图标统一线性风格
- 不混用多套 UI 语言

## 禁止项

不要使用：

- 玻璃拟态
- 大面积渐变
- 彩色阴影
- 发光效果
- 花哨动画
- 多主色混用
- Hero 区或营销页式包装

## 预览卡片

- 最终导出尺寸固定：`1080 × 1440`
- 预览可缩放，但必须保持严格等比
- 卡片背景保持干净，不加纹理、插画或装饰背景

## 一句话定义

这是一个**以内容生产和结果预览为核心的浅色专业工具界面**。
